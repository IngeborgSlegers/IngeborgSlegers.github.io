About Me

I was born and raised in Belgium. Belgium is a tri-lingual country and I am multilingual. I moved to Indiana for further education and studied Experience Design and Archaeology, followed by a Master's Degree in Strategic Leadership and Organizational Design. I fell in love with coding because of my affinity for learning languages and allows me to work creatively while still using my logic driven brain.

I am incredibly passionate about creating ways to help others through app creation and website design to emphasize the user experience. My work is data-driven, innovative, grounded in mindfulness. I am looking forward to projects that prioritize information sharing and strengthen a sense of community.

My High School education taught me to learn with my head, understand with my heart, and apply with my hands. I approach coding with this philosophy and strongly attempt to embody it in my everday life.

I make applications with heart. I want my work to help and to make people feel loved.

Projects



Jardin Naps
Jardin Naps is an introductory application into Angular (using Angular 7). This team project was created during the red Badge (Weeks 9 - 12) in a 24 hour sprint at Eleven Fifty Academy.



Nest
Nest is a full stack application using PostgreSQL, Express, Angular 7, and Node. It was styled using Angular Material.

Nest is an easy place for searching roommates to find one another. Users have the ability to create or login with an account and create listings, message other users, and interact with admin. Features include full CRUD, an admin portal, authentication, password encryption, and database associations. This team project was created during the red Badge (Weeks 9 - 12) in a 2 week sprint at Eleven Fifty Academy.

Dèjá Brew
Dèjá Brew is a full stack application using PostgreSQL, Express, React, and Node. It was styled using Reactstrap.

Users have the ability to create an account and add different teas into a list. Features include full CRUD, session validation, password encryption, and foreign keys/associations. This project was created during the Blue Badge (Weeks 4 - 8) in a 2 week sprint at Eleven Fifty Academy.
